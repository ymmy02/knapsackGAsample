number of items : (Int)
weight : (Int)
weight : (Int)
...
weight : (Int)
value : (Int)
value : (Int)
...
value : (Int)

```

number of items : 5
weight : 3
weight : 10
weight : 7
weight : 2
weight : 11
value : 3
value : 9
value : 10
value : 3
value : 15
```
